To run the project, first connect all hardware components properly to the Arduino Uno based on the circuit setup, including the RFID module, LCD I2C display, arcade buttons, LEDs, buzzer, sound sensor, and control buttons. Next, open the Arduino IDE and install the required libraries such as MFRC522, SPI, Wire, and LiquidCrystal_I2C. After that, upload the provided code to the Arduino Uno and ensure the correct board and COM port are selected. Once uploaded, power the system using USB or an external power source, then press the power/start button to turn it on. The LCD will prompt the user to scan an RFID card; once scanned, the system will be activated and the player can press the start button to begin the game. During gameplay, LEDs will light up randomly and the player must press the correct corresponding button quickly to earn points. If the player hits the wrong button or fails to respond in time, the game will end, and they can restart using the retry button or view the highest score using the top score button.
